Static

Static is the default value. An element with the position static; is not positioned in any special way. A static element is said to be NOT postioned and an element with its position set to anything else is said to be positioned static.

Relative

Setting the top, bottom, left, and right properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

Relative properties

A fixed element is positioned relative to the viewport, which means it always stays in the same place EVEN if the page is scrolled. As with relative, the top, right, bottom, and left properties are used.

fixed

A fixed element is positioned relative to the viewport, which means it always stays in the same place EVEN if the page is scrolled. As with relative, the top, right, bottom, and left properties are used.
Absolute is the trickiest position value. Absolute behaves like fixed except relative to the nearest postioned ancestor instead of relative to the viewport. If an absolutely-postioned element has no positioned ancestors, it uses the document body, and still moves along with the page scrolling. Remember, a "positioned" element is one whose position is anything except static.
